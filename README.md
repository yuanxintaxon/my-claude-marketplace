# liyuanxin-tools

Personal Claude Code plugin marketplace.

## Available Plugins

| Plugin | Description |
|--------|-------------|
| `flutter-getx-toolkit` | Scaffolding skills, review agents, and architecture rules for Flutter + GetX projects |

## Install

```bash
# Register this marketplace
/plugin marketplace add liyuanxin/my-claude-marketplace

# Install a plugin
/plugin install flutter-getx-toolkit@liyuanxin-tools
```
